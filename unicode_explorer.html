Codepoints
?
A codepoint is Unicode's globally unique ID for every character — written as U+XXXX. Every script on earth has its own reserved range. The Odia block is U+0B00–U+0B7F.

Grapheme Clusters
?
What a human perceives as one visual unit. In Indic scripts, a consonant + Virama + consonant = 1 cluster (e.g., ର୍ପ), even though that's 3 Unicode codepoints.

Raw Bytes (UTF-8)
?
How the text is physically stored on disk or in memory. ASCII = 1 byte. Indic scripts (Odia, Devanagari, etc.) = 3 bytes per codepoint. Rare characters = 4 bytes.

NFC vs NFD
?
Two ways to represent the same text. NFC = precomposed (é as 1 cp). NFD = decomposed (e + ◌́ as 2 cp). For Odia they're usually equal — critical to normalize before OCR evaluation to avoid false CER inflation.
